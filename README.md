# 数字記憶テスト（逆唱）

数字を音声で提示し、逆順で入力してもらう実験用の単純なウェブアプリです。参加者IDの入力、練習、本試行、結果CSVのダウンロードまでブラウザだけで完結します。

## セットアップ
- 追加のビルドやライブラリは不要です。`index.html` をブラウザで開くだけで動作します（Chrome/Edge/Safari 最新版を推奨）。
- 音声ファイルを `audio/digits/0.mp3` 〜 `9.mp3` と `audio/beep.wav` として配置してください。`app.js` の `CONFIG.audio` でパスを変えられます。

## 使い方
1. `index.html` をブラウザで開く。
2. 参加者IDを入力して「練習に進む」を押す。
3. 練習（3桁×3本固定）を行うか、「練習をスキップ」で本試行へ進む。
4. ビープ音のあとに流れる数字を聞き、終了後に表示が変わったら逆順で入力して送信（Enter でも可）。入力ごとに自動で次試行へ進みます。
5. 本試行（3〜9桁、各9本）が終わると結果画面が表示されます。「結果をCSVで保存」でファイルをダウンロードできます。
6. 「最初に戻る」で参加者ID入力画面に戻り、別の参加者で再開できます。

## 出力CSVの列
- `participant_id`: 参加者ID
- `mode`: タスクモード（固定で `backward`）
- `trial`: 試行番号（1始まり）
- `level`: 桁数
- `target`: 提示した数字列
- `expected`: 正答の逆順
- `response`: 入力された数字列
- `correct`: 正誤（1/0）
- `rt_ms`: 反応時間（ms）

## 補足設定
- 桁数や試行数などは `app.js` の `CONFIG` で変更できます（デフォルト: 本試行は3〜9桁・各9本、音声間隔なども調整可能）。
- 練習で流す数字列は `CONFIG.practiceDigits` に配列で指定できます。
